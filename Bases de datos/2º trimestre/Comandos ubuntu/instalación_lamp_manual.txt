Servidor mysql 
apt-get install mysql-server mysql-client  
(REPASO: Ver el estado y la versión)


	*El paquete Ubuntu mysql-client incluye las siguientes herramientas de línea de comandos (y más):

	mysql : el cliente de línea de comandos mysql para ejecutar sentencias SQL.
	mysqladmin - cliente para administrar un servidor MySQL.
	mysqldump - un programa de copia de seguridad de base de datos. El comando mysqldump escribe el contenido de las tablas de la base de datos en archivos de texto que puede usar para restaurar las bases de datos.
	mysqlreport : genera un informe amigable de los valores de estado importantes de MySQL.
	mysqlcheck : un cliente de línea de comandos para verificar, reparar y optimizar tablas.
