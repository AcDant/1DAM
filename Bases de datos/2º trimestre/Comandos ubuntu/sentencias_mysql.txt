IMPORTANTE EL PUNTO Y COMA

show databases;
	Muestra las bases de datos que hay activas. Al comienzo se mostrarán las BD que utiliza el propio sistema, se almacenan metadatos (tablas, datos propios de MySQL). 

select database();
	Comprobamos que BD está en uso

select schema();
	Hace lo mismo

use nombreDeBD;
	Te permite seleccionar la BD que quieres utilizar
	Si vuelves a ejecutar select database(); te mostrará el nombre de la BD que has puesto en nombreDeBD

show tables;
	Ver las tablas que hay en una BD. REQUISITO: estar en una BD (use BD).

describe nombreDeTabla;
	Donde nombreDeTabla es una tabla de tu BD
	Te mostrará la definición de una tabla (sus columnas, el tipo de datos, si admite nulos, PK, etc.)

create database nombreDeBD;
	Esta sentencia te permitirá crear una BD con el nombre que establezcas en nombreDeBD 

drop database nombreDeBD;
	Eliminará la BD con el nombre que se establezca en nombreDeBD

Se puede añadir: create database IF NOT EXISTS nombreDeBD; 


select user();
	Mirar el usuario que está en uso

select session_user() y select system_user(); (Hacen lo mismo)

select version();
	Mirar la versión

select now();
	Muestra día y fecha actual del sistema

select charset('holita');
	Devuelve la codificación de caracteres de la cadena, si no existe cadena devuelve null.

select charset(convert('holita' using latin1))
	Forzamos a que se utilice latin1 para que CHARSET devuelva distinto de utf8mb4.

show character set;
	Devuelve las codificaciones de caracteres disponibles en mysql.

current role();
	Devuelve el rol del usuario actual.

current_user();
	Devuelve el usuario autenticado y el host.



OTRAS FORMAS DE PONER SENTENCIAS SQL

select version(), now();
	Muestra en dos columnas la información que hemos puesto separada por una coma. 

**Ponerlo en varias líneas.

select version(); select now();
	Se ejecutan como dos sentencias diferentes, ya que hay un ; que separa ambas sentencias.
